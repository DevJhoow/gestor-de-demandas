# Gestor de Demandas

Sistema desenvolvido para simular o fluxo de solicitações entre áreas de negócio e equipes de tecnologia.

## Objetivo

Permitir o cadastro, priorização e acompanhamento de demandas, simulando um ambiente corporativo onde áreas de negócio solicitam melhorias e novas funcionalidades para a equipe de TI.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades Planejadas

- Cadastro de demandas
- Definição de prioridade
- Controle de status
- Dashboard de acompanhamento
- Persistência de dados no navegador
